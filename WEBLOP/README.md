# WEBLOP - Web Loop Video Creator

ブラウザで動作するループ動画作成ツールのプロトタイプです。
FFmpeg.wasm を使用してクライアントサイドで動画処理を行います。

## 起動方法

```bash
cd WEBLOP
npm start
```

ブラウザで http://localhost:3000 を開いてください。

## 機能

- 動画ファイルのドラッグ&ドロップ
- リアルタイムプレビュー
- ループ回数設定（1-20回）
- ループモード選択
  - **Reverse**: 正再生→逆再生→正再生... (1234→321→234→321)
  - **Forward**: 正再生の繰り返し (1234→1234→1234)
- 品質選択（高/標準/高速）
- ループ動画のダウンロード

## 技術仕様

- **動画処理**: FFmpeg.wasm 0.12.x
- **対応形式**: MP4, WebM, MOV, AVI
- **出力形式**: MP4 (H.264)

## 制限事項

- **SharedArrayBuffer必須**: Cross-Origin-Isolationが必要なため、ローカルサーバー経由でアクセスする必要があります
- **初回ロード**: FFmpeg.wasmのダウンロード（約25MB）が必要
- **処理速度**: ネイティブアプリより遅い（2-10倍程度）
- **メモリ制限**: 大きな動画ファイルはブラウザのメモリ制限に達する可能性があります
- **推奨ファイルサイズ**: 50MB以下

## ブラウザ対応

- Chrome 92+
- Firefox 90+
- Edge 92+
- Safari 15.2+

※ SharedArrayBuffer をサポートするブラウザが必要です
